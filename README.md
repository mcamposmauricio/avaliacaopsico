# MarQ HR — Avaliação Psicossocial

Plataforma de avaliação de fatores psicossociais no trabalho conforme a NR-1. Gestão de campanhas, análises e relatórios de compliance.

## Stack

- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui
- Vite

## Funcionalidades

- Estrutura organizacional (unidades, departamentos, cargos)
- Cadastro de colaboradores
- Campanhas de pesquisa psicossocial
- Runtime de questionário anônimo com consentimento LGPD
- Scoring automatizado por dimensão
- Alertas de risco
- Plano de ação
- Relatórios e governança
- Multi-tenant com RLS

## Desenvolvimento

```sh
npm install
npm run dev
```
